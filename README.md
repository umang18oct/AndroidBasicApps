# AndroidBasicApps


## About Topper Displayer App

### App is composed of two activities named MainActivity(Parent) and DisplayActivity(Child).The MainActivity’s UI has text fields and a button . The user enters year. The year is checked against the first two characters of regno ,when the user hits the button, the event handler OnClick(View) finds the details of top 5 rank holders ,and passes the information to DisplayActivity ,which then displays the rank holders.. The DisplayActivity’s UI has TextViews for displaying regno ,name and CGPA in descending order of CGPA
#### CSV file details : REGNO, NAME, CGPA
